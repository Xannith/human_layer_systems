What it unlocks
The levels map onto the prompt library tiers. Crawl is single task execution. Walk is multi input synthesis built for genuine two turn work. Run is building a capability you can hand off: a full pipeline prompt, an agent even a clumsy one, or a tool like this ladder. Fly is orchestration: a dashboard you operate that drives many single purpose agents, some chained into pipelines, with one agent watching the others and reporting back. Nobody reaches Fly without mastering the foundation, and nobody flies without guidance. The next version adds a tool skin that maps the agents onto a specific product's features for a partner channel.
