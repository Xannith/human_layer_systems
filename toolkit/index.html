Tools by lifecycle phase

Awareness

Epistemic Debt Demonstrator

Use when a client thinks a better model is the answer. Shows that a bleeding corpus produces a confidently wrong assistant -- same model, same question, different corpus, different truth.

 Adoption to Run

Prompt Evolution Ladder

Use to show how a prompt grows from one line to an orchestration, and where agents and tools fit. Crawl, Walk, Run are one animal getting stronger. Fly is a different animal entirely.

 Implementation

Workflow Redesign

Use to show where AI enters a real process and where the human stays. The role does not shrink -- it moves up to the steps that carry judgment, money, and accountability.

 Augment, Automate

Trust Dial

Use when the question is how much to let AI do. Autonomy is per task and depends on stakes and readiness -- not one setting you flip for the whole company.

 Optimization

Telemetry and the Loop

Use to show how the system watches itself and feeds the next cycle. Optimization does not end the work -- the findings open the next Awareness cycle.
